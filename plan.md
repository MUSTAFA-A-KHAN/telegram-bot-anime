1. **Create an Image Generation Service**
   - Create a new file `service/image_generator.go`.
   - Implement a function `GenerateLeaderboardImage(client, collection string, chatID int64, title string)` that queries MongoDB for the top 10 players, formats their rank, name (with emojis), and score, and draws this onto an in-memory image.
   - The image should have a nice background color, headers for Rank, Player, and Score, and rows for the top 10 players.
   - Use the standard `golang.org/x/image/font/gofont/goregular` font or basic Go fonts to draw the text, as we don't have TTF files in the repository. We will load `goregular.TTF`.

2. **Add Telegram Bot Commands and Callbacks**
   - In `controller/bot.go` and `controller/categoryBot/categoryBot.go`, update the `/statsimage` and `/statsimageglobal` command handlers to send inline buttons:
     - "Word Guess Image Group", "Wordle Image Group", "Scramy Image Group"
     - "Word Guess Image Global", "Wordle Image Global", "Scramy Image Global"
   - Handle the corresponding callbacks (e.g., `statsimagegroup_wordguess`, `statsimageglobal_wordle`) in both `bot.go` and `categoryBot.go`.

3. **Handle sending the image**
   - In the callback handlers, call the `GenerateLeaderboardImage` function.
   - If successful, use `tgbotapi.NewPhotoUpload` with `tgbotapi.FileBytes` to send the generated byte slice back to the user.
   - Clean up the callback loading state by answering it.
